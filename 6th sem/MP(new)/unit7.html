Group B
How do you find a view element? Which dialog boxes can you use?

Finding a view: In Java, you use findViewById(R.id.element_id). In modern Android, you might use View Binding or Data Binding to access views more safely without findViewById.

Common Dialog Boxes:

  • AlertDialog: Displays a message and up to three buttons.
  • DatePickerDialog: Used to select a date.
  • TimePickerDialog: Used to select a time.
  • ProgressDialog: (Now deprecated) Displays progress of an operation.
  • BottomSheetDialog: Slides up from the bottom of the screen.
Group B
What are different data types supported by SQLite? How to decode JSON data?

SQLite Data Types:

  • NULL: The value is a NULL value.
  • INTEGER: The value is a signed integer (1, 2, 3, 4, 6, or 8 bytes).
  • REAL: The value is a floating-point value.
  • TEXT: The value is a text string (UTF-8, UTF-16BE, or UTF-16LE).
  • BLOB: The value is a blob of data, stored exactly as it was input.

JSON Decoding Example:

String jsonString = "{ \"name\": \"John\", \"age\": 25 }";
try {
    JSONObject obj = new JSONObject(jsonString);
    String name = obj.getString("name");
    int age = obj.getInt("age");
} catch (JSONException e) {
    e.printStackTrace();
}
Group B
How to generate a signed APK? Write a program to locate the user's current location.

Generating Signed APK:

  1. In Android Studio, go to Build > Generate Signed Bundle / APK.
  2. Select APK and click Next.
  3. Choose an existing Key Store or create a new one.
  4. Enter the passwords and alias, then click Next.
  5. Select the build variant (usually release) and click Finish.

Locate Current Location (Java):

// Manifest Permissions
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />

// LocationActivity.java
FusedLocationProviderClient client = LocationServices.getFusedLocationProviderClient(this);
client.getLastLocation().addOnSuccessListener(location -> {
    if (location != null) {
        double lat = location.getLatitude();
        double lon = location.getLongitude();
        Toast.makeText(this, "Lat: " + lat + " Lon: " + lon, Toast.LENGTH_SHORT).show();
    }
});
Group C
Provided that a SQLite database named "College" with a table named Student... Develop an application to insert 5 student records and display them.

DBHelper.java

public class DBHelper extends SQLiteOpenHelper {
    public DBHelper(Context context) { super(context, "College.db", null, 1); }
    public void onCreate(SQLiteDatabase db) {
        db.execSQL("CREATE TABLE Student (Roll INTEGER PRIMARY KEY, Name TEXT, Address TEXT)");
    }
    public void onUpgrade(SQLiteDatabase db, int oldV, int newV) { db.execSQL("DROP TABLE IF EXISTS Student"); onCreate(db); }

    public void insertStudent(int roll, String name, String addr) {
        SQLiteDatabase db = this.getWritableDatabase();
        ContentValues cv = new ContentValues();
        cv.put("Roll", roll); cv.put("Name", name); cv.put("Address", addr);
        db.insert("Student", null, cv);
    }

    public Cursor getAllStudents() {
        return this.getReadableDatabase().rawQuery("SELECT * FROM Student", null);
    }
}
