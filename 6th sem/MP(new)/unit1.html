Group B
What do you mean by mobile device? List and explain any four popular programming languages used for developing mobile applications.

A mobile device is a small, hand-held computing device, typically having a display screen with touch input and/or a miniature keyboard. It is designed for portability and allows users to access information, communicate, and perform computing tasks on the go. Examples include smartphones, tablets, and personal digital assistants (PDAs).

Popular Programming Languages for Mobile Development:

  1. Java: Historically the primary language for Android development. It is object-oriented, robust, and has a vast ecosystem of libraries. It runs on the Dalvik Virtual Machine (DVM) or Android Runtime (ART).
  2. Kotlin: Now the preferred language for Android development (officially supported by Google). It is concise, expressive, and fully interoperable with Java, while reducing common programming errors like null pointer exceptions.
  3. Swift: Developed by Apple for iOS, macOS, watchOS, and tvOS. It is designed to be fast, modern, and safe. It replaced Objective-C as the primary language for iOS apps.
  4. Dart: Used by the Flutter framework (developed by Google) for building cross-platform applications. It allows developers to write one codebase that runs on both Android and iOS with high performance.
Group B
What is a smart mobile device? Explain different platforms used in mobile devices.

A smart mobile device is an electronic device that is generally connected to other devices or networks via different wireless protocols such as Bluetooth, NFC, Wi-Fi, 5G, etc., that can operate to some extent interactively and autonomously. It features an advanced operating system and allows the installation of third-party applications.

Common Mobile Platforms:

  • Android (Google): An open-source, Linux-based platform used by a wide variety of manufacturers (Samsung, Google, Xiaomi, etc.). It holds the largest market share globally.
  • iOS (Apple): A closed-source platform exclusive to Apple devices (iPhone, iPad). It is known for its high security, smooth performance, and premium ecosystem.
  • Windows Phone / Windows 10 Mobile: (Now deprecated) Microsoft's attempt at a mobile OS that featured a "tile" based interface and integration with the Windows ecosystem.
  • HarmonyOS (Huawei): A distributed operating system designed for a wide range of devices, developed in response to restrictions on Android usage.
Group B
What are different mobile application development approaches? Explain in detail.

There are four main approaches to mobile application development:

  1. Native Apps: Developed specifically for one platform (e.g., Swift for iOS, Kotlin for Android).
    • Pros: Best performance, full access to hardware (GPS, Camera), best user experience.
    • Cons: Higher cost (need separate codebases for iOS and Android), requires platform-specific expertise.
  2. Web Apps (Mobile Web): Essentially websites designed to look and feel like apps, accessed via a browser.
    • Pros: Single codebase, easy maintenance, no app store approval needed.
    • Cons: Limited access to hardware, requires internet, slower performance.
  3. Hybrid Apps: A mix of native and web apps. They are built using web technologies (HTML, CSS, JS) and wrapped in a native container (like Cordova or Ionic).
    • Pros: Single codebase, faster development than native, access to some hardware.
    • Cons: Performance is not as smooth as native, can have UI inconsistencies.
  4. Cross-Platform Apps: Built using a single codebase that compiles into native code (e.g., Flutter, React Native).
    • Pros: Near-native performance, single codebase for multiple platforms, cost-effective.
    • Cons: May lag behind native platforms in supporting latest OS features.
Group B
What is Android? Explain different categories of mobile devices.

Android is a mobile operating system based on a modified version of the Linux kernel and other open-source software, designed primarily for touchscreen mobile devices such as smartphones and tablets. It is developed by a consortium of developers known as the Open Handset Alliance and commercially sponsored by Google.

Categories of Mobile Devices:

  • Smartphones: Multi-purpose mobile computing devices that combine a mobile phone with features like a web browser, camera, and the ability to run software applications.
  • Tablets: Larger than smartphones, typically featuring 7 to 13-inch screens. Optimized for media consumption, browsing, and light productivity.
  • Wearables: Small devices worn on the body, such as Smartwatches (Apple Watch, Galaxy Watch) and fitness trackers.
  • PDAs (Personal Digital Assistants): Handheld devices that act as a personal organizer (mostly evolved into modern smartphones).
  • E-Readers: Devices designed primarily for reading digital books (e.g., Amazon Kindle).
Group B
What is a mobile device? Explain different features of a mobile device.

A mobile device is a handheld computer with a touch-sensitive screen or keyboard and a wireless connection to the internet.

Key Features of Mobile Devices:

  • Portability: Designed to be carried easily in a pocket or bag.
  • Wireless Connectivity: Support for Wi-Fi, Bluetooth, 4G/5G, and GPS.
  • Touch Interface: Most modern devices use capacitive touchscreens for user interaction.
  • Sensors: Equipped with sensors like Accelerometer (orientation), Gyroscope, Magnetometer, and Proximity sensors.
  • Power Source: Powered by rechargeable batteries (Lithium-ion).
  • Multimedia Support: Built-in cameras, speakers, and microphones for audio and video tasks.
