Why is fragment needed in android application? How can a fragment be added to an activity?

A Fragment is a modular section of an activity, which has its own lifecycle, receives its own input events, and which you can add or remove while the activity is running. Fragments are needed for:

  • Reusability: Use the same fragment in different activities.
  • Dynamic UI: Change UI components based on device orientation or screen size (e.g., multi-pane layouts on tablets).
  • Modularization: Dividing complex activity code into smaller, manageable pieces.

Ways to Add a Fragment:

  1. Statically in XML: Declaring the fragment within the activity's layout file using the <fragment> tag.
  2. Dynamically via Java/Kotlin: Using FragmentManager and FragmentTransaction to add, replace, or remove fragments at runtime.

What is the difference between activity and fragment?
Feature Activity Fragment
Existence Can exist independently. Must be hosted by an activity.
Lifecycle Managed by the OS. Managed by the host activity's lifecycle.
Purpose Represents a single screen with a UI. Represents a portion of a UI or behavior.
Context It is a Context itself. Needs getActivity() or getContext().
Group B
What is Toast? Develop an android application to create an alert dialog to input any number and then calculate the factorial of the number and display the result in Toast.

A Toast is a simple feedback about an operation in a small popup. It only fills the amount of space required for the message and the current activity remains visible and interactive.

Factorial with Alert Dialog and Toast:

// MainActivity.java Snippet
public void showFactorialDialog() {
    AlertDialog.Builder builder = new AlertDialog.Builder(this);
    builder.setTitle("Factorial Calculator");
    final EditText input = new EditText(this);
    input.setInputType(InputType.TYPE_CLASS_NUMBER);
    builder.setView(input);

    builder.setPositiveButton("Calculate", (dialog, which) -> {
        int n = Integer.parseInt(input.getText().toString());
        long fact = 1;
        for (int i = 1; i <= n; i++) fact *= i;
        Toast.makeText(this, "Factorial: " + fact, Toast.LENGTH_LONG).show();
    });
    builder.show();
}

Group C
Develop an android application to calculate simple interest using a customize dialog box.

Custom Dialog Layout (dialog_interest.xml)

<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent" android:layout_height="wrap_content"
    android:orientation="vertical" android:padding="16dp">
    <EditText android:id="@+id/etP" android:hint="Principal" android:inputType="numberDecimal" />
    <EditText android:id="@+id/etT" android:hint="Time (years)" android:inputType="numberDecimal" />
    <EditText android:id="@+id/etR" android:hint="Rate (%)" android:inputType="numberDecimal" />
</LinearLayout>

Java Code to Show Dialog:

public void showInterestDialog() {
    View dialogView = getLayoutInflater().inflate(R.layout.dialog_interest, null);
    EditText etP = dialogView.findViewById(R.id.etP);
    EditText etT = dialogView.findViewById(R.id.etT);
    EditText etR = dialogView.findViewById(R.id.etR);

    new AlertDialog.Builder(this)
        .setTitle("Calculate Simple Interest")
        .setView(dialogView)
        .setPositiveButton("Calculate", (dialog, which) -> {
            double p = Double.parseDouble(etP.getText().toString());
            double t = Double.parseDouble(etT.getText().toString());
            double r = Double.parseDouble(etR.getText().toString());
            double si = (p * t * r) / 100;
            Toast.makeText(this, "Interest: " + si, Toast.LENGTH_LONG).show();
        }).show();
}
Group C
Develop an android application with a context menu having items "Red", "Yellow", "Green" etc. and change the background color of the layout when selected.
// MainActivity.java
LinearLayout mainLayout;

@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_main);
    mainLayout = findViewById(R.id.mainLayout);
    registerForContextMenu(mainLayout); // Long press anywhere to show menu
}

@Override
public void onCreateContextMenu(ContextMenu menu, View v, ContextMenu.ContextMenuInfo info) {
    super.onCreateContextMenu(menu, v, info);
    menu.setHeaderTitle("Select Background Color");
    menu.add(0, v.getId(), 0, "Red");
    menu.add(0, v.getId(), 0, "Yellow");
    menu.add(0, v.getId(), 0, "Green");
}

@Override
public boolean onContextItemSelected(MenuItem item) {
    if (item.getTitle().equals("Red")) mainLayout.setBackgroundColor(Color.RED);
    else if (item.getTitle().equals("Yellow")) mainLayout.setBackgroundColor(Color.YELLOW);
    else if (item.getTitle().equals("Green")) mainLayout.setBackgroundColor(Color.GREEN);
    return true;
}
