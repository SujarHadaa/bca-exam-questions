Unit 5: URLConnections

Reading the Header Using URLConnection

Describe the process of reading the Header using URLConnection.

Reading the response header of a web resource is a common task in network programming, used to get metadata about the response before processing the actual content. The process using Java's URLConnection class involves these steps:

  1. Create a URL Object: First, you need to create a URL object representing the target resource.
  2. Open a Connection: Call the openConnection() method on the URL object. This returns a URLConnection object (or a subclass like HttpURLConnection).
  3. Establish the Connection: Although some header fields might be available before connecting, explicitly calling connect() or getting the input stream (which implicitly connects) ensures that the HTTP request is sent and the headers are received from the server.
  4. Retrieve Header Fields: Use the methods of the URLConnection object to access the header fields. You can either get specific, well-known headers using dedicated methods or iterate through all of them.

There are two main ways to get header fields:

  • Using specific getter methods: For common headers like Content-Type, Content-Length, and Date, URLConnection provides direct methods like getContentType(), getContentLengthLong(), and getDate().
  • Using a generic method: You can retrieve any header by its name using getHeaderField(String name). To get all headers, you can use getHeaderFields() which returns a Map>, or you can loop using getHeaderFieldKey(int n) and getHeaderField(int n).

Java Program to Read All Response Headers

This program connects to a URL and prints out all the response headers sent by the server.


import java.io.IOException;
import java.net.URL;
import java.net.URLConnection;
import java.util.List;
import java.util.Map;

public class AllHeaderReader {

    public static void main(String[] args) {
        try {
            URL url = new URL("https://www.google.com");
            URLConnection connection = url.openConnection();

            // The connection is implicitly established when getting header fields

            System.out.println("All Response Headers from " + url + ":");
            System.out.println("--------------------------------------------------");

            // Get all header fields as a Map
            Map> headerFields = connection.getHeaderFields();

            // Iterate over the map and print each header
            for (Map.Entry> entry : headerFields.entrySet()) {
                String key = entry.getKey();
                List values = entry.getValue();
                
                // The status line has a null key
                if (key == null) {
                    System.out.println("Status Line: " + String.join(", ", values));
                } else {
                    System.out.println(key + ": " + String.join(", ", values));
                }
            }

        } catch (IOException e) {
            System.err.println("Error connecting to the URL or reading headers.");
            e.printStackTrace();
        }
    }
}

Key Points to Remember: Reading Headers with URLConnection

  • Step 1: Create a URL object.
  • Step 2: Call openConnection() on the URL.
  • Step 3: Use methods like getHeaderField(String name) or getHeaderFields() to get the headers.
  • No explicit connect needed: Calling a get-header method implicitly establishes the connection.
  • Header Map: getHeaderFields() returns a map where the key is the header name and the value is a list of header values.
