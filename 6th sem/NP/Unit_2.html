Unit 2: Internet Addresses

The InetAddress Class

What is the use of the InetAddress class?

The InetAddress class in Java is used to represent an Internet Protocol (IP) address. It provides methods to resolve hostnames to IP addresses and vice versa. This class is fundamental for network programming as it encapsulates both IPv4 and IPv6 addresses, allowing you to identify and connect to other machines on a network.

Key uses include:

  • Getting the IP address of a local machine or a remote host.
  • Converting a hostname (like "www.google.com") into an IP address.
  • Checking the type of IP address (IPv4 or IPv6).

Key Points to Remember: InetAddress Class

  • Represents an IP address (both IPv4 and IPv6).
  • Resolves hostnames to IP addresses.
  • Provides methods to get local and remote host information.

The NetworkInterface Class

What is the use of the NetworkInterface class?

The NetworkInterface class represents a network interface, which is a point of connection between a computer and a network. This can be a physical device like an Ethernet card or a Wi-Fi adapter, or a virtual one. It is primarily used to retrieve information about the local network interfaces, such as their names, IP addresses, and MAC (Media Access Control) addresses.

Basic Features of the NetworkInterface class:

  • Listing Network Interfaces: You can get a list of all network interfaces available on the system.
  • Getting Interface Information: For each interface, you can retrieve its name (e.g., "eth0", "wlan0"), display name, and a list of associated IP addresses.
  • Retrieving MAC Address: It provides a method to get the hardware address (MAC address) of the interface, which is crucial for uniquely identifying a device on a local network.
  • Checking Interface Status: You can check if an interface is up and running, if it's a loopback interface, or if it supports multicast.

Key Points to Remember: NetworkInterface Class

  • Represents a network interface (e.g., Ethernet card).
  • Used to get information about local network connections.
  • Key feature: Provides access to the MAC address.

Java Program to Show IP, MAC, and IP Version

This program demonstrates how to use the InetAddress and NetworkInterface classes to get the system's IP address, check if it's IPv4 or IPv6, and find the corresponding MAC address.


import java.net.InetAddress;
import java.net.NetworkInterface;
import java.net.SocketException;
import java.net.UnknownHostException;

public class NetworkInfo {

    public static void main(String[] args) {
        try {
            // Get the local host's IP address
            InetAddress localHost = InetAddress.getLocalHost();
            System.out.println("IP Address: " + localHost.getHostAddress());

            // Check if the address is IPv4 or IPv6
            if (localHost instanceof java.net.Inet4Address) {
                System.out.println("IP Version: IPv4");
            } else if (localHost instanceof java.net.Inet6Address) {
                System.out.println("IP Version: IPv6");
            } else {
                System.out.println("IP Version: Unknown");
            }

            // Get the network interface for the local host
            NetworkInterface networkInterface = NetworkInterface.getByInetAddress(localHost);
            if (networkInterface != null) {
                // Get the MAC address
                byte[] macAddressBytes = networkInterface.getHardwareAddress();
                if (macAddressBytes != null) {
                    // Format the MAC address for display
                    StringBuilder macAddress = new StringBuilder();
                    for (int i = 0; i < macAddressBytes.length; i++) {
                        macAddress.append(String.format("%02X%s", macAddressBytes[i], (i < macAddressBytes.length - 1) ? "-" : ""));
                    }
                    System.out.println("MAC Address: " + macAddress.toString());
                } else {
                    System.out.println("MAC Address: Not found or not accessible.");
                }
            } else {
                System.out.println("Network interface for the IP address not found.");
            }

        } catch (UnknownHostException e) {
            System.err.println("Error: Could not resolve the local host name.");
            e.printStackTrace();
        } catch (SocketException e) {
            System.err.println("Error: Could not access the network interface.");
            e.printStackTrace();
        }
    }
}

Key Points to Remember: IP/MAC Address Program

  • Use InetAddress.getLocalHost() to get your computer's IP address.
  • Use instanceof java.net.Inet4Address to check for IPv4.
  • Use NetworkInterface.getByInetAddress() to get the network card for that IP.
  • Use getHardwareAddress() on the network interface to get the MAC address.
  • Remember to handle UnknownHostException and SocketException.
