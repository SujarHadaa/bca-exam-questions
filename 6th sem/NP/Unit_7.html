Unit 7: Socket for Servers

Server-Side Program for a Daytime Service

What is a Daytime Service?

A daytime service is a simple network service that, when a client connects, sends the current date and time to the client and then closes the connection. It's a classic example used to demonstrate server-side socket programming.

How to Implement a Server-Side Daytime Service

The implementation involves creating a ServerSocket that waits for client connections. When a client connects, the server accepts the connection, which creates a regular Socket for communication. The server then writes the current date and time to the socket's output stream and closes the connection.

Java Program for a Daytime Server

This program creates a server that listens on port 1313. When a client connects, it sends the current date and time and then closes the connection.


import java.io.IOException;
import java.io.OutputStreamWriter;
import java.io.PrintWriter;
import java.net.ServerSocket;
import java.net.Socket;
import java.util.Date;

public class DaytimeServer {

    public static final int PORT = 1313;

    public static void main(String[] args) {
        try (ServerSocket serverSocket = new ServerSocket(PORT)) {
            System.out.println("Daytime Server started on port " + PORT);
            System.out.println("Waiting for clients to connect...");

            while (true) { // Keep the server running indefinitely
                try (Socket clientSocket = serverSocket.accept()) { // Wait for a client connection
                    System.out.println("Client connected from: " + clientSocket.getInetAddress());

                    // Get the output stream to send data to the client
                    PrintWriter out = new PrintWriter(new OutputStreamWriter(clientSocket.getOutputStream()));

                    // Get the current date and time
                    String dateTime = new Date().toString();

                    // Send the date and time to the client
                    out.println(dateTime);
                    out.flush(); // Ensure the data is sent

                    System.out.println("Sent date and time to the client: " + dateTime);
                    // The client socket is automatically closed by the try-with-resources statement
                }
            }
        } catch (IOException e) {
            System.err.println("Error with the server socket.");
            e.printStackTrace();
        }
    }
}

How to Test the Daytime Server

You can test this server using a simple client program or even a command-line tool like telnet or netcat. For example, in your terminal, you would run:

telnet localhost 1313

After connecting, the server will send the date and time and then close the connection.

Key Points to Remember: Daytime Server

  • Create a ServerSocket: This waits for incoming connections on a specific port.
  • Loop and Accept Connections: Use an infinite loop and serverSocket.accept() to handle multiple clients, one after another.
  • Get Output Stream: Use clientSocket.getOutputStream() to send data to the client.
  • Send Data: Write the current date and time to the output stream.
  • Close the Socket: It's crucial to close the client socket after serving the request to free up resources. Using a try-with-resources statement is the best practice.
