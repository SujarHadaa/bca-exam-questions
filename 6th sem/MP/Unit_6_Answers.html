Unit 6: Listview, Gridview and Recyclerview

How is List View differ from Recycler View? Explain with an example.

ListView and RecyclerView are both fundamental UI components in Android for displaying scrollable lists of items. However, RecyclerView was introduced as a more advanced and efficient successor to ListView.

Key Differences:

  • View Recycling: This is the most significant difference. RecyclerView recycles and reuses cells (views) as the user scrolls. When a list item scrolls off the screen, its view is not destroyed. Instead, it's repurposed for a new item scrolling onto the screen. This drastically improves performance and reduces memory consumption, preventing lag with long or complex lists. ListView, by contrast, can be configured to recycle views, but it's not as efficient or enforced by default.
  • ViewHolder Pattern: RecyclerView enforces the use of the ViewHolder design pattern. A ViewHolder is an object that stores a reference to the subviews within an item's layout (e.g., TextViews, ImageViews). This avoids the need to repeatedly call `findViewById()` for each item, which is a performance-intensive operation. While the ViewHolder pattern is a recommended practice for ListView, it is not mandatory.
  • LayoutManager: RecyclerView delegates the responsibility of positioning items to a `LayoutManager`. This provides great flexibility, allowing you to easily implement vertical lists (`LinearLayoutManager`), horizontal lists, grids (`GridLayoutManager`), and staggered grids (`StaggeredGridLayoutManager`) without changing your adapter. ListView only supports a single, vertical layout.
  • Item Animator: RecyclerView has a built-in, extensible system (`ItemAnimator`) for animating item additions, removals, and reordering. Implementing similar animations in ListView is a much more complex, manual process.

RecyclerView Example:

1. Add RecyclerView dependency (in `build.gradle`):

implementation 'androidx.recyclerview:recyclerview:1.3.0'

2. Layout for the list (`activity_main.xml`):

<?xml version="1.0" encoding="utf-8"?>
<androidx.recyclerview.widget.RecyclerView
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:id="@+id/recycler_view"
    android:layout_width="match_parent"
    android:layout_height="match_parent" />

3. Layout for a single list item (`list_item.xml`):

<?xml version="1.0" encoding="utf-8"?>
<TextView xmlns:android="http://schemas.android.com/apk/res/android"
    android:id="@+id/item_text"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:padding="16dp"
    android:textSize="18sp" />

4. Create the Adapter (`MyAdapter.java`):

public class MyAdapter extends RecyclerView.Adapter<MyAdapter.MyViewHolder> {
    private String[] data;

    public MyAdapter(String[] data) {
        this.data = data;
    }

    @NonNull
    @Override
    public MyViewHolder onCreateViewHolder(@NonNull ViewGroup parent, int viewType) {
        View view = LayoutInflater.from(parent.getContext())
                .inflate(R.layout.list_item, parent, false);
        return new MyViewHolder(view);
    }

    @Override
    public void onBindViewHolder(@NonNull MyViewHolder holder, int position) {
        holder.textView.setText(data[position]);
    }

    @Override
    public int getItemCount() {
        return data.length;
    }

    public static class MyViewHolder extends RecyclerView.ViewHolder {
        public TextView textView;
        public MyViewHolder(View view) {
            super(view);
            textView = view.findViewById(R.id.item_text);
        }
    }
}

5. Set up the RecyclerView in `MainActivity.java`:

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        RecyclerView recyclerView = findViewById(R.id.recycler_view);
        recyclerView.setLayoutManager(new LinearLayoutManager(this));

        String[] myData = {"Apple", "Banana", "Cherry", "Date", "Elderberry", "Fig", "Grape"};
        MyAdapter adapter = new MyAdapter(myData);
        recyclerView.setAdapter(adapter);
    }
}

Develop an android application to display 8 programming languages in ListView.

Full Code Example:

1. `activity_main.xml`:

<?xml version="1.0" encoding="utf-8"?>
<ListView
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:id="@+id/list_view"
    android:layout_width="match_parent"
    android:layout_height="match_parent" />

2. `MainActivity.java`:

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        ListView listView = findViewById(R.id.list_view);

        String[] languages = {"Java", "Kotlin", "Swift", "Dart", "Python", "JavaScript", "C++", "C#"};

        // ArrayAdapter is a simple adapter for displaying arrays of data
        ArrayAdapter<String> adapter = new ArrayAdapter<>(
                this, // Context
                android.R.layout.simple_list_item_1, // Layout for a single row
                languages // Data source
        );

        listView.setAdapter(adapter);

        // Optional: Set an item click listener
        listView.setOnItemClickListener((parent, view, position, id) -> {
            String selectedLanguage = languages[position];
            Toast.makeText(MainActivity.this, "You selected: " + selectedLanguage, Toast.LENGTH_SHORT).show();
        });
    }
}
