Unit 5: UI Fragments, Menus and Dialogs

Why is fragment needed in android application? How can a fragment be added to an activity?

Why Fragments are Needed:

Fragments represent a reusable portion of your app's UI. A fragment defines and manages its own layout, has its own lifecycle, and can handle its own input events. They are essential for creating flexible and adaptive UIs that can be used across a variety of screen sizes, like phones and tablets. Key benefits include:

  • Modularity: You can break down a complex activity into smaller, reusable, and more manageable pieces.
  • Reusability: A single fragment can be used in multiple activities.
  • Adaptability: You can design multi-pane layouts for tablets while using the same fragments in a single-pane layout for phones.

Adding a Fragment to an Activity:

You can add a fragment to an activity either statically in the XML layout file or dynamically using Java/Kotlin code.

1. Static Addition (in XML):

You use the <fragment> tag directly in your activity's layout file.

activity_main.xml: 
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical">

    <fragment
        android:id="@+id/my_static_fragment"
        android:name="com.example.myapp.MyFragment"
        android:layout_width="match_parent"
        android:layout_height="match_parent" />

</LinearLayout>

2. Dynamic Addition (in Java/Kotlin):

You use the FragmentManager to add, remove, or replace fragments at runtime. This provides more flexibility.

activity_main.xml (with a container for the fragment): 
<?xml version="1.0" encoding="utf-8"?>
<FrameLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:id="@+id/fragment_container"
    android:layout_width="match_parent"
    android:layout_height="match_parent" />
MainActivity.java: 
public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        if (savedInstanceState == null) {
            getSupportFragmentManager().beginTransaction()
                .setReorderingAllowed(true)
                .add(R.id.fragment_container, MyFragment.class, null)
                .commit();
        }
    }
}

What is menu? Develop an android application to create a popup menu having menu item "Edit", "Copy", "Paste", "Redo" and "Undo.".

What is a Menu:

A menu is a common UI component that presents a list of options to the user. Android provides a framework for several types of menus:

  • Options Menu: The primary collection of menu items for an activity, typically placed in the app bar.
  • Context Menu: A floating menu that appears when a user performs a long-press on a view.
  • Popup Menu: A modal menu anchored to a specific view, useful for providing an overflow of actions.

Popup Menu Example:

1. Define the layout (e.g., activity_main.xml):

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:padding="16dp">

    <Button
        android:id="@+id/show_popup_menu_button"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_centerInParent="true"
        android:text="Show Popup Menu" />

</RelativeLayout>

2. Create a menu resource file (e.g., res/menu/popup_menu.xml):

<?xml version="1.0" encoding="utf-8"?>
<menu xmlns:android="http://schemas.android.com/apk/res/android">
    <item android:id="@+id/menu_edit" android:title="Edit" />
    <item android:id="@+id/menu_copy" android:title="Copy" />
    <item android:id="@+id/menu_paste" android:title="Paste" />
    <item android:id="@+id/menu_redo" android:title="Redo" />
    <item android:id="@+id/menu_undo" android:title="Undo" />
</menu>

3. In your activity, create and show the popup menu:

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        Button button = findViewById(R.id.show_popup_menu_button);
        button.setOnClickListener(v -> {
            PopupMenu popup = new PopupMenu(MainActivity.this, v);
            popup.getMenuInflater().inflate(R.menu.popup_menu, popup.getMenu());

            popup.setOnMenuItemClickListener(item -> {
                int itemId = item.getItemId();
                if (itemId == R.id.menu_edit) {
                    Toast.makeText(MainActivity.this, "Edit clicked", Toast.LENGTH_SHORT).show();
                    return true;
                } else if (itemId == R.id.menu_copy) {
                    Toast.makeText(MainActivity.this, "Copy clicked", Toast.LENGTH_SHORT).show();
                    return true;
                } else if (itemId == R.id.menu_paste) {
                    Toast.makeText(MainActivity.this, "Paste clicked", Toast.LENGTH_SHORT).show();
                    return true;
                } else if (itemId == R.id.menu_redo) {
                    Toast.makeText(MainActivity.this, "Redo clicked", Toast.LENGTH_SHORT).show();
                    return true;
                } else if (itemId == R.id.menu_undo) {
                    Toast.makeText(MainActivity.this, "Undo clicked", Toast.LENGTH_SHORT).show();
                    return true;
                }
                return false;
            });
            popup.show();
        });
    }
}

Develop an android application to display an alert dialog box.

Full Code Example:

1. Define the layout (e.g., activity_main.xml):

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:padding="16dp">

    <Button
        android:id="@+id/show_alert_button"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_centerInParent="true"
        android:text="Show Alert" />

</RelativeLayout>

2. In your activity, create and show the AlertDialog:

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        Button showAlertButton = findViewById(R.id.show_alert_button);
        showAlertButton.setOnClickListener(v -> {
            showAlertDialog();
        });
    }

    private void showAlertDialog() {
        AlertDialog.Builder builder = new AlertDialog.Builder(this);
        builder.setTitle("Alert");
        builder.setMessage("This is a standard alert dialog.");
        builder.setIcon(android.R.drawable.ic_dialog_alert);

        builder.setPositiveButton("OK", (dialog, which) -> {
            Toast.makeText(getApplicationContext(), "OK button clicked", Toast.LENGTH_SHORT).show();
            dialog.dismiss();
        });

        builder.setNegativeButton("Cancel", (dialog, which) -> {
            Toast.makeText(getApplicationContext(), "Cancel button clicked", Toast.LENGTH_SHORT).show();
            dialog.dismiss();
        });
        
        AlertDialog dialog = builder.create();
        dialog.show();
    }
}
