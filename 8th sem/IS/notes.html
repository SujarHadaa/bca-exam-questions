Information Security - Notes

Unit I: Overview of Computer security

This unit introduces the fundamental concepts of computer security, including the CIA triad (Confidentiality, Integrity, Availability), threats, attacks, and assets. It also covers security design principles, attack surfaces, and the development of a computer security strategy.

Unit II: Cryptographic Algorithms

This unit delves into the world of cryptography. It covers classical cryptosystems like Caesar and Vigenere ciphers, modern ciphers (block vs. stream, symmetric vs. asymmetric), and the details of symmetric encryption (DES, AES) and asymmetric encryption (Diffie-Hellman, RSA). It also touches upon the mathematical foundations of cryptography, such as number theory and Galois fields.

Unit III: Message Authentication and Hash Functions

This unit focuses on ensuring the integrity and authenticity of messages. It covers message authentication, hash functions (MD4, MD5, SHA-1), HMAC (Hash-based Message Authentication Code), and digital signatures.

Unit IV: User Authentication

This unit explores the different ways to authenticate users. It covers the principles of user authentication, password-based authentication, token-based authentication, biometric authentication, remote user authentication, and two-factor authentication.

Unit V: Access Control

This unit deals with controlling access to resources. It covers access control principles, subjects, objects, and access rights. It also discusses different access control models like Discretionary Access Control (DAC), Role-Based Access Control (RBAC), and Attribute-Based Access Control (ABAC).

Unit VI: Malicious Software and Intrusion

This unit covers the threats posed by malicious software and intrusions. It discusses different types of malware like viruses, worms, and Trojan horses. It also covers intrusion detection systems (IDS), their analysis approaches (anomaly-based vs. signature-based), and the use of honeypots.

Unit VII: Network Security

This unit focuses on securing networks. It provides an overview of network security and covers important security protocols like S/MIME, PGP, SSL/TLS, and IPSec. It also discusses the role of firewalls in network security.

Unit VIII: Security Auditing

This unit covers the process of security auditing. It discusses the importance of security audits, the architecture of a security audit, the security audit trail, and the implementation of logging functions for audit trail analysis.
